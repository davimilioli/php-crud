## Neste repositório ensino a fazer um CRUD simples com PHP

### O que significa CRUD?
#### Resumo

> É um acrônimo para Create(criar), Read(ler), Update(atualizar) e Delete(apagar).

#### Create

> Criação de dados

#### Read

> Consultar dados

#### Update

> Editar Dados

#### Delete

> Deletar dados

## Comandos Básicos CRUD

- Inserindo dados no banco de dados

  `INSERT INTO nome_da_tabela (nome_da_coluna) VALUES ('nome_do_dado');`

- Selecionando dados no banco dados

  `SELECT * FROM nome_da_tabela;`

- Atualizando dados no banco de dados

  `UPDATE nome_da_tabela SET nome_da_coluna = 'novo_valor' WHERE id = valor_do_id;`

- Deletando dados no banco de dados

  `DELETE FROM nome_da_tabela WHERE nome_da_coluna = 'nome_do_dado;`
